AgriSetu

AgriSetu is a full-stack web application designed to support farmers by providing a platform to buy agricultural products, rent machinery, and access AI-based guidance.

Features

- 👨‍🌾 Farmer Dashboard
- 🛒 Buy agricultural products (seeds, fertilizers, etc.)
- 🚜 Rent farming equipment
- 🤖 AI-based farming assistance (coming soon)
- 🔐 User authentication (JWT आधारित login system)
- 🧑‍🤝‍🧑 Role-based system (Farmer, Supplier, Equipment Owner)



Tech Stack

Frontend:
- React.js
- Tailwind CSS
- React Router

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

Other:
- JWT Authentication
- bcrypt (password hashing)
